创建GitHub仓库

首先，我们需要在GitHub上创建一个新的仓库。

  1. 登录到 GitHub。
  2. 在右上角点击你的头像，然后选择“Your repositories”。
  3. 点击“New”按钮，创建一个新的仓库。
  4. 填写仓库名称（如 my-new-project），添加一个描述（可选）。
  5. 选择公开（Public）或私有（Private）仓库。
  6. 勾选“Initialize this repository with a README”。
  7. 点击“Create repository”按钮。
