1793. 好子数组的最大分数

题目

给你一个整数数组 nums （下标从 0 开始）和一个整数 k 。

一个子数组 (i, j) 的 分数 定义为 min(nums[i], nums[i+1], ..., nums[j]) * (j - i + 1) 。一个 好 子数组的两个端点下标需要满足 i <= k <= j 。

请你返回好子数组的最大可能 分数 。

示例

示例1

输入：nums = [1,4,3,7,4,5], k = 3
输出：15
解释：最优子数组的左右端点下标是 (1, 5) ，分数为 min(4,3,7,4,5) (5-1+1) = 3 5 = 15 。

示例2

输入：nums = [5,5,4,5,4,1,1,1], k = 0
输出：20
解释：最优子数组的左右端点下标是 (0, 4) ，分数为 min(5,5,4,5,4) (4-0+1) = 4 5 = 20 。

提示

1 <= nums.length <= 105
1 <= nums[i] <= 2 * 104
0 <= k < nums.length

解题思路

这道题首先要求是一个包含下标k元素的子数组，这意味着最终返回的子数组中必定有数字nums[k]。
然后开始分析分数，由题意得 分数 = 子数组内元素数量 * 子数组内最小的元素。
这里考虑采用双指针，首先left和right指向nums[k]，然后扩大范围，寻找子数组最小值为nums[k]的子数组范围，然后依次寻找小于nums[k]的子数组的范围,同时更新返回值。
这道题的重点是 先锚定一个子数组的最小值，然后再经可能的扩大子数组内元素的数量，然后再确定下一个子数组的最小值，再扩大范围，依次重复，直到整个数组都被囊括。
